# ИНСТРУКЦИЯ ПО ОФОРМЛЕНИЮ ОТЧЕТА

## Шаг 1: Конвертация Markdown в DOCX

Для конвертации файла `ОТЧЕТ_КУРСОВОЙ.md` в формат DOCX (Word) используйте один из способов:

### Способ 1: Использование Pandoc (рекомендуется)

1. Установите Pandoc: https://pandoc.org/installing.html
2. Выполните команду:
```bash
pandoc ОТЧЕТ_КУРСОВОЙ.md -o ОТЧЕТ_КУРСОВОЙ.docx --reference-doc=reference.docx
```

> **Примечание:** Для создания reference.docx с правильным форматированием по ГОСТ, создайте документ с нужными стилями (Заголовок 1, Заголовок 2, Обычный текст) и используйте его как шаблон.

### Способ 2: Использование онлайн-конвертеров

- https://www.markdowntopdf.com/
- https://dillinger.io/ (экспорт в DOCX)
- https://www.convertio.co/ru/md-docx/

### Способ 3: Использование Microsoft Word

1. Откройте Microsoft Word
2. Файл → Открыть → выберите `ОТЧЕТ_КУРСОВОЙ.md`
3. Word автоматически конвертирует Markdown в формат Word
4. Сохраните как DOCX

## Шаг 3: Форматирование по ГОСТ 7.32-2017

### Общие требования:
- Шрифт: Times New Roman, 14pt (основной текст), 12pt (таблицы, примечания)
- Интервал: 1,5 (основной текст), 1 (таблицы, формулы)
- Поля: левое — 30мм, правое — 15мм, верхнее — 20мм, нижнее — 20мм
- Нумерация страниц: сквозная, внизу страницы по центру
- Абзацный отступ: 1,25 см

### Заголовки:
- Заголовок 1: жирный, 16pt, выравнивание по центру
- Заголовок 2: жирный, 14pt, выравнивание по левому краю
- Заголовок 3: жирный, 14pt, выравнивание по левому краю, отступ 1,25 см

### Таблицы:
- Нумерация: сквозная с указанием раздела (например, "Таблица 3.1")
- Заголовок таблицы: над таблицей, жирный, выравнивание по левому краю
- Подпись таблицы: под таблицей (если нужна)

### Рисунки (снимки экрана):
- Нумерация: сквозная с указанием раздела (например, "Рисунок 5.1")
- Подпись: под рисунком, выравнивание по центру

### Код:
- Шрифт: Courier New, 10-12pt
- Фон: светло-серый (если возможно)
- Перенос строк должен быть сохранен

### Ссылки на источники:
- В квадратных скобках: [1], [2, с. 45], [3]

2. Название файла: `Курсовой_проект_БД_[Ваша_Фамилия].docx`
3. Отправьте на электронную почту преподавателя с достойно оформленным письмом
4. Тема письма: "Курсовой проект по дисциплине 'Проектирование баз данных' - [Ваша Фамилия], [Группа]"
5. Текст письма:
```
